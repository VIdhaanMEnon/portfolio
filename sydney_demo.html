This is a browser showcase of Sydney’s product experience. It simulates how the assistant hears, interprets, and executes commands without needing local installs or OS permissions.

In this web demo, browser-safe actions like opening websites and searches will work in a new tab. Local desktop actions remain simulated.

Why this demo exists

The real Sydney project runs locally with Python, automation modules, OCR, audio tooling, and system access. For a portfolio site, this simulated version is the cleanest way to communicate the vision instantly.
